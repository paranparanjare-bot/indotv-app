# INDOTV 📺

Simple IPTV player for Indonesia TV channels.

## Features
- M3U playlist loader (default: Indonesia channels)
- Custom URL input — paste any M3U link
- Channel search
- Auto-hide channel name overlay (3 seconds)
- Fullscreen video playback
- D-Pad/remote friendly (Android TV compatible)
- Lightweight, no external dependencies

## Build
Built automatically via GitHub Actions. Download from Actions → Artifacts.
